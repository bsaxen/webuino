Sketches:

In order for Simuino to load the sketch, it has to contain the following lines:

  • // SKETCH_NAME:  your_sketch_name 
  • // BOARD_TYPE: UNO (or MEGA)

The sketch also need to have scenario information. i . e:

  • //  SCENDIGPIN  (pin) (step) (value)
  • // SCENANAPIN  (pin) (step) (value)

Example:

// SCENDIGPIN 9 1 0

// SCENDIGPIN 9 40 1

// SCENDIGPIN 10 100 0

// SCENDIGPIN 10 200 1

// SCENANAPIN 4 1 5

// SCENANAPIN 4 80 12

// SCENANAPIN 5 1 8

// SCENANAPIN 5 120 18
